# 资治通鉴网页系统

## 系统概述

本系统是一个基于HTML/CSS/JavaScript的资治通鉴网页系统，旨在提供一个美观、易用的平台，展示资治通鉴原文、白话文翻译以及不同AI的分析内容。系统采用中国风设计风格，支持响应式布局，适配各种设备。

### 核心功能

- **完整内容整合**：无删减展示所有AI思考内容
- **智能格式转换**：自动处理Markdown到HTML的转换
- **中国风保持**：完美保留传统设计风格
- **灵活配置**：支持单个项目和批量处理
- **易于使用**：清晰的文档和快速启动脚本

## 快速启动

### 方法一：使用快速启动脚本

1. **运行脚本**：双击 `start_server.ps1` 文件
2. **选择操作**：
   - 选项1：启动主目录服务器
   - 选项2：启动指定项目服务器
   - 选项3：批量处理所有项目
   - 选项4：退出脚本

### 方法二：手动启动本地服务器

1. **打开命令提示符**或**PowerShell**
2. **导航到项目目录**：
   ```powershell
   cd e:\zztjaljbz\zztjaljbz
   ```
3. **启动Python HTTP服务器**：
   ```powershell
   python -m http.server 8000
   ```
4. **访问网页**：在浏览器中打开 `http://localhost:8000`

## 使用方法

### 访问首页

- 打开 `http://localhost:8000` 即可访问首页
- 首页展示三个模块：资治通鉴精读、医案精读、身高评估
- 点击对应卡片进入相应模块

### 资治通鉴精读

1. **进入资治通鉴目录**：点击首页的"资治通鉴精读"卡片
2. **选择专题**：在资治通鉴目录页选择感兴趣的专题
3. **查看内容**：
   - 点击"原文"标签查看资治通鉴原文
   - 点击"白话文"标签查看白话文翻译
   - 点击各AI分析标签查看不同AI的分析内容

### 医案精读

- 点击首页的"医案精读"卡片进入医案精读页面

### 身高评估

- 点击首页的"身高评估"卡片进入身高评估工具

## 添加新内容

### 添加新的资治通鉴专题

1. **创建专题文件夹**：在 `zztj` 目录下创建新的专题文件夹，例如 `新专题`
2. **准备内容文件**：在新文件夹中创建以下文件：
   - `原文.md`：资治通鉴原文
   - `白话文.md`：白话文翻译
   - `chatgpt.md`：ChatGPT分析
   - `deepseek.md`：DeepSeek分析
   - `gemini.md`：Gemini分析
   - `grok.md`：Grok分析
   - `kimi.md`：Kimi分析
   - `千问.md`：千问分析
   - `智谱.md`：智谱分析
3. **创建index.html**：复制其他专题的index.html文件到新文件夹，修改标题和内容引用
4. **更新资治通鉴目录**：编辑 `zztj/index.html`，添加新专题的链接

## 常见问题与解决方案

### 问题：本地服务器无法启动

**解决方案**：
1. 确保已安装Python 3.6+
2. 检查端口8000是否被占用
3. 尝试使用其他端口，例如：
   ```powershell
   python -m http.server 8080
   ```

### 问题：网页显示不完整

**解决方案**：
1. 清除浏览器缓存
2. 检查Markdown文件格式是否正确
3. 确保所有必要的文件都已创建

### 问题：AI分析内容未显示

**解决方案**：
1. 检查对应AI的markdown文件是否存在
2. 检查index.html中是否正确引用了该文件
3. 确保文件内容格式正确
