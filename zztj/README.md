# 国学智慧与健康网站

一个简约中国风网站，包含资治通鉴精读、医案精读和身高评估三个主要部分。

## 网站结构

### 1. 资治通鉴精读
- 智氏兴衰与赵氏崛起
- 智伯灭族
- 豫让之义与赵氏之乱

### 2. 医案精读
- 经典医案解析
- 名医经验传承
- 养生保健指导

### 3. 身高评估
- 儿童身高预测
- 生长发育监测
- 营养与运动指导

## 技术特点

- 🏮 中国风设计：传统配色、书法字体、古典纹样
- 📱 响应式布局：支持桌面和移动设备
- 🎨 现代动画：平滑过渡、渐入效果
- 🔧 语义化HTML：结构清晰，易于维护

## 本地运行

1. 克隆仓库
2. 直接打开 `index.html` 文件即可

## 部署

可部署到任何静态网站托管服务，如：
- GitHub Pages
- Vercel
- Netlify
- 阿里云OSS
- 腾讯云COS
